# UM-Spectrum-Login
This is a tool to login UM Spectrum by using python requests

## Prerequisites:
+ python3
  + re
  + requests

## How to use:
1. clone the code into your pc
2. edit the ```login.py``` file, change line 5 - line 6 with your own information
3. run the code
